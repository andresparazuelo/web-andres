# Portfolio base

Base estatica para tu portfolio personal.

## Estructura propuesta

- Hero con propuesta de valor clara
- Sobre mi con resumen profesional
- Experiencia con formato timeline
- Educacion
- Casos de estudio o proyectos
- Marca personal y publicaciones
- Contacto

## Siguiente iteracion recomendada

1. Sustituir el email placeholder y el enlace de LinkedIn.
2. Añadir una foto tuya y logos reales de empresas.
3. Convertir cada bloque de proyectos en un caso de estudio con metricas.
4. Refinar textos con el contenido exacto del CV.

## Uso

Abre `portfolio/index.html` en el navegador.
